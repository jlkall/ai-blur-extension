CloseAI

Detect and blur AI-generated content with advanced machine learning

What is CloseAI?

CloseAI is a browser extension that automatically detects and blurs AI-generated text and images using advanced machine learning models. It helps you identify synthetic content as you browse the web, showing confidence ratings so you know how certain the detection is.

How It Works

CloseAI uses an ensemble of machine learning techniques to detect AI-generated content:

Detection Methods

  • Perplexity analysis using DistilGPT2
  • Burstiness and sentence variance detection
  • N-gram repetition pattern analysis
  • Word entropy and frequency analysis
  • Image metadata and URL pattern detection
  • Visual artifact and color banding detection
  • Texture smoothness analysis
  • Edge consistency and sharpness detection

Privacy & Security

CloseAI respects your privacy:

  • All processing happens locally in your browser
  • No data is collected or transmitted
  • No external network requests
  • No user tracking or analytics
  • Open source and transparent

Installation

CloseAI is available on the Chrome Web Store. Simply install it and start browsing - AI-generated content will be automatically detected and blurred with confidence ratings.

How It Looks

When CloseAI detects AI-generated content, it:

  • Blurs the text or image
  • Shows a confidence rating badge (e.g., "85.43%")
  • Allows you to click to reveal the content

Open Source

CloseAI is open source and available on GitHub. You can view the code, contribute, or report issues.
